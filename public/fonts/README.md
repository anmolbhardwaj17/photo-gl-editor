# Custom Fonts

Add your custom font files here.

## Supported Formats
- `.woff2` (recommended)
- `.woff`
- `.ttf`
- `.otf`

## File Naming
Name your font files descriptively, for example:
- `Inter-Regular.woff2`
- `Inter-Bold.woff2`
- `CustomFont-Regular.woff2`

## Next Steps
After adding your font files, we'll:
1. Define `@font-face` rules in CSS
2. Add the fonts to the Tailwind config
3. Apply them to the app
